## 🚀 Обновление версии 0.54.2
_Дата релиза: 21 апреля 2026_

### ✨ Улучшения

- **Добавлена дополнительная мера безопасности для Duress Mode**
  При входе в `duress mode` теперь можно автоматически очистить адресную книгу.

- **Добавлена поддержка нескольких провайдеров для сети TRON**

- **Добавлены фоновые уведомления о новых транзакциях**
  Кошелёк теперь может отправлять уведомления о новых транзакциях, даже когда приложение работает в фоне.

- **Улучшено отображение дерева токенов**

- **Улучшено отображение потерь при обмене**

- **Улучшен показ предупреждения о нехватке монет для комиссии**
  Плашка о недостаточном балансе нативной монеты сети теперь отображается только после завершения синхронизации блокчейна и только если фактический баланс действительно равен нулю.

### 🐛 Исправления

- **Стабилизирована работа сценариев с кошельками Monero, Bitcoin и Tangem**
  Аккаунты, созданные или восстановленные из Monero seed-фразы, теперь корректно распознаются только как Monero-кошельки.
  На экране повторной отправки Bitcoin вместо пустого красного блока теперь показывается понятное сообщение об ошибке.
  Также повышена общая стабильность сценариев работы с картами Tangem.

- **Исправлены проблемы с отображением и статусом незавершённого обмена**
  Сообщение `Обмен не завершился` теперь отображается полностью, а статус транзакции определяется корректно.

- **Незавершённые двухэтапные обмены теперь отображаются только в нужном кошельке**
  Баннер `Unfinished swap` и список незавершённых обменов больше не показываются в других кошельках.

- **Улучшен пользовательский опыт на экране отправки при временно недоступном адаптере**
  Экран теперь корректно обрабатывает ситуацию, когда адаптер кошелька ещё не успел восстановиться после возврата приложения из фона или повторной инициализации.

- **Исправлено дублирование поля суммы получения в форме транзакции**

- **Исправлен краш при открытии справочного экрана по info-иконке**

- **Исправлена работа клавиатуры при вводе альтернативной суммы на экране отправки**

- **Приведено к единому виду название поля провайдера в истории обменов**

- **Исправлен визуальный артефакт в диалоге подтверждения создания нового Monero-адреса**

- **Восстановлена кнопка обмена XEC на странице актива**

- **Исправлена ложная ошибка `Нет интернета` после ручного обновления рыночных виджетов**

- **Устранено кратковременное дублирование исходящих транзакций в истории**
  В истории больше не появляются временные дубликаты отправки из-за `pending transaction`.

- **Удалён лишний разделитель в запросе к Blockchain API**

## 🚀 Обновление версии 0.54.0
_Дата релиза: 1 апреля 2026_

### ✨ Улучшения

- **Добавлена защита от Address Poisoning (подмены адресов)**
  Подозрительные транзакции теперь помечаются красной иконкой, а их данные скрываются.
  В приложении введена классификация транзакций:
  `созданные в данном кошельке`, `от адресов из адресной книги`, `полученные из блокчейна` и `подозрительные (скам)`.

- **Добавлена возможность ротации адресов Monero для вашей безопасности**

- **Добавлена возможность скрывать все транзакции в разделе транзакций**
  Работает аналогично механике скрытия баланса по нажатию.

- **Добавлена информация о необходимости монет сети для отправки и работы с токенами**
  Если у пользователя недостаточно нативных монет сети для оплаты комиссии, приложение теперь показывает это явно.

### 🐛 Исправления

- **Исправлена ошибка обмена Jetton при использовании аппаратных карт Tangem**

- **Исправлена ошибка выбора оптимального провайдера для обмена**
