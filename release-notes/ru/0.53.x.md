## 🚀 Обновление версии 0.53.0
_Дата релиза: 26 марта 2026_

### ✨ Улучшения

- **Добавлена возможность выполнять сложные обмены через двух провайдеров**
  Теперь приложение поддерживает multi-hop обмены через промежуточный актив.
  Например, обмен `PIRATE Jetton` на `PIRATE BEP-20` выполняется по цепочке `PIRATE Jetton -> TON -> PIRATE BEP-20`.

- **Доработана логика отображения оставшегося числа блоков после повторного запуска приложения**

- **Удалена команда отмены транзакции из интерфейса**
  Эта команда вызывала путаницу у пользователей, поскольку транзакцию в блокчейне фактически отменить нельзя.

### 🐛 Исправления

- **Исправлено поведение плашки стейкинга для PIRATE**
  Активация в пуле теперь корректно отображается начиная от `100 PIRATE`.

- **Исправлена проблема с сетью TRON из-за ошибок HTTP 429 со стороны Tron Explorer**

- **Исправлено зависание синхронизации ZEC при переключении между кошельками**

- **Исправлен краш при открытии окна выбора монеты**
